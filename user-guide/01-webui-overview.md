# 01. 화면 구성과 WebUI 사용법

## WebUI 주소

현재 Nanobot WebUI 는 아래 주소에서 사용한다.

```text
http://127.0.0.1:8765/webui/
```

같은 Mac 안에서 접속하는 로컬 주소다. 외부 공개용 주소가 아니다.

## 기본 화면

WebUI 는 크게 세 영역으로 나뉜다.

- 왼쪽 사이드바: 채팅 목록, 대시보드, 설정
- 가운데 채팅 영역: 선택한 세션의 대화 내용
- 상단/상태 영역: 현재 타깃, 채널, 연결 상태, 상태 요약 줄, 세부 정보

용어가 헷갈리면 [10. WebUI 용어 기준](10-webui-terminology.md) 을 함께 본다.

## 대시보드

대시보드는 전체 상태를 한눈에 보는 첫 화면이다.

주로 아래 내용을 확인하는 용도다.

- 최근 채팅 흐름
- 진행 중이거나 막힌 작업
- 승인 대기 상태
- 자동화 결과 요약
- 현재 작업 상태

채팅을 시작하려면 사이드바에서 기존 채팅을 고르거나 새 채팅을 만든다.

## 채팅 목록

왼쪽 목록에는 WebUI 채팅과 Telegram 채팅이 함께 보일 수 있다.

각 항목에는 보통 아래 정보가 붙는다.

- 최근 메시지 미리보기
- 채널 표시
- model target 표시
- 승인 대기 badge

Telegram 채팅은 일반 WebUI 채팅과 달리 실제 Telegram 앱의 대화와 연결된다.

또 하나 알아둘 점은 아래와 같다.

- WebUI 내부 websocket 채팅은 삭제 메뉴가 보일 수 있다.
- Telegram 같은 bridged 채팅은 WebUI 에서 삭제 버튼을 숨기는 정책을 따른다.

## 상태 요약 줄

헤더 아래에는 한 줄짜리 상태 요약 줄이 보일 수 있다.

예를 들어 아래 정보가 짧게 합쳐져 표시된다.

- 관련 세션 수
- 현재 채널
- 최근 활동 시각

이 줄은 긴 상태 카드를 여러 개 펼치는 대신, 현재 대화와 연결된 맥락을 빠르게 읽기 위한 요약 표면이다.

## 메시지 입력

아래 입력창에 메시지를 쓰고 Enter 를 누르면 전송된다.

```text
Enter: 전송
Shift+Enter: 줄바꿈
```

일반 질문과 명령어를 모두 입력할 수 있다.

예시:

```text
현재 상태를 알려줘
/calendar
/model list
```

## 설정

설정에서는 WebUI 표시와 기본 모델 관련 상태를 확인한다.

현재 작업된 기능 기준으로 중요한 항목은 아래와 같다.

- 테마 전환
- 채팅 글자 크기 조절
- 기본 공급자와 기본 모델 확인
- 런타임에서 잠긴 설정 확인
- 추론 표시 수준 조절

추론 표시 수준은 보통 아래 의미로 이해하면 된다.

- 끄기: reasoning 관련 줄을 숨긴다.
- 상태만: 진행 상태만 짧게 본다.
- 요약: 답변 앞 reasoning 요약을 짧게 본다.
- 디버그 추적: 더 자세한 trace 를 본다.

## 세부 정보

세부 정보는 현재 답변이나 자동화 결과의 추가 맥락을 확인하는 공간이다.

예를 들어 작업 개요, 현재 작업, 연결된 외부 세션, action result, blocked 상태 같은 내용을 자세히 볼 수 있다.

상단의 `세부 정보` 버튼을 눌러 열며, 항상 본문 위에 큰 카드를 유지하기보다 필요할 때만 펼쳐서 보는 흐름을 기준으로 한다.

## 초보자용 사용 순서

1. WebUI 를 연다.
2. 왼쪽에서 채팅을 선택한다.
3. 먼저 `/status` 또는 `현재 상태를 알려줘` 를 입력한다.
4. 일정이 궁금하면 `/calendar` 또는 `/calendar today` 를 입력한다.
5. 모델을 바꾸고 싶으면 `/model list` 를 입력한다.

## 알아두면 좋은 점

- WebUI 는 단순 표시 화면이 아니라 Telegram 세션에도 답장을 보낼 수 있다.
- Telegram 채팅에서 보낸 slash command 결과도 WebUI 에 남도록 보강되어 있다.
- action result 는 기본적으로 compact row 로 보이고, 자세한 내용은 row 클릭 또는 `세부 정보` 로 펼친다.
- reasoning 이 켜져 있으면 assistant 답변 앞에 `생각` 또는 `생각 중` 줄이 보일 수 있다.
- WebUI 가 이상해 보이면 브라우저 새로고침 전에 gateway health 와 bootstrap 을 확인하는 것이 좋다.
