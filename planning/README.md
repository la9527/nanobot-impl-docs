# Nanobot Planning Docs

이 디렉터리는 현재 구현 우선순위와 다음 제품 slice 를 정리하는 planning 공간이다.

## 포함 내용

- [todo.md](./todo.md): 지금 남은 active validation 과 next candidate
- [execution-backlog/README.md](./execution-backlog/README.md): 구현 slice 별 workplan
- [concepts/README.md](./concepts/README.md): 개인비서 제품 축의 conceptual reference

## 운영 원칙

- active queue 는 `todo.md` 에만 둔다.
- 구현 상세 범위는 `execution-backlog/` 에서 관리한다.
- conceptual 초안은 직접 체크리스트로 쓰지 않고, 필요할 때 backlog 문서로 승격한다.
