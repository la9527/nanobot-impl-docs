# Nanobot Archive Docs

이 디렉터리는 현재 active source of truth 는 아니지만, 보존 가치가 있는 설계/검토/운영 기록을 모은다.

## 포함 내용

- `assistant-direction-a/`: 초기 방향 A 설계와 workplan archive
- `observable-reasoning-plan/`: reasoning visibility 관련 설계 기록
- `calendar-webui-reliability-review/`: Calendar/WebUI reliability 검토 기록
- `webui-ux-redesign/`: WebUI 정보 구조와 density 재설계 검토
- `upstream-main-sync-2026-05-08/`: sync/merge 시점 운영 기록

## 운영 원칙

- 이 디렉터리 문서를 active TODO 로 직접 사용하지 않는다.
- 다시 구현을 열 때는 필요한 부분만 `planning/todo.md` 와 새 backlog 문서로 승격한다.
